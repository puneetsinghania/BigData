" Comparative Analysis of the Historical Snowfall Patterns using Big Data and ArcGIS"


Every year, snowfalls and snowstorms cost millions when it comes to loss of life and property damage. This causes disruption and disturbance into the functioning of the nation. Multiple accidents, blizzards are costly for the same reason. In order to avoid this, we need to come with preventive majors. One such thing which can help us is, analyzing the weather data which is collected by the weather stations around the globe on a regular basis. This data is present in enormous amounts but one such drawback is that it is difficult to analyze and extract meaningful information from it using the traditional data analysis techniques. We strongly believe that using the modern visualization techniques we can provide insights to take preventive measures/actions in order to tackle climate change. Going ahead, we have provided a detailed breakdown of how exactly we have come up with our solution and what different techniques/ methodologies we have used to do so.
